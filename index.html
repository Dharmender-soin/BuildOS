TRDI Progress Framework & Live Validator

Adjust the values below to verify how progress ratios recalculate and validation rules trigger:

✅ TRDI hierarchy matches constraints ($T \ge R \ge D \ge I$)
Release Progress (R/T)74.1%
Delivery Progress (D/T)55.6%
Installation Progress (I/T)46.3%
1,400 Pending Release (T - R)
1,000 Pending Delivery (R - D)
500 Pending Installation (D - I)
TRDI Validation Hierarchy Rule:

T ≥ R ≥ D ≥ I. Installed cannot exceed Delivered. Delivered cannot exceed Released. Released cannot exceed Total.
